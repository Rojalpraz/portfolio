
## üöÄ Deployment

This portfolio is deployed on Vercel. To deploy your own version:

1. Push your code to GitHub
2. Import the repository on [Vercel](https://vercel.com)
3. Configure build settings (usually auto-detected)
4. Deploy!

For manual deployment:
```bash
npm run build
npm run start
```

## üéØ Customization

To customize this portfolio for your own use:

1. **Update Personal Information**: Edit the content component files
2. **Add Your Projects**: Update the projects section with your own work
3. **Modify Color Scheme**: Adjust colors in the CSS/styling files
4. **Replace Images**: Add your own photos and project screenshots in `/public/`
5. **Update Resume**: Replace the resume file with your own
6. **Social Links**: Update social media links in the contact section
